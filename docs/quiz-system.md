# 答题系统规则

## 数学挑战

- 题目数量：100 道
- 运算类型：20以内加减法
- 时间限制：10 分钟
- 基础奖励：
  - 满分(100题)：300 蛋币
  - 95+ 题：200 蛋币
  - 80+ 题：150 蛋币
  - 60+ 题：100 蛋币
  - 40+ 题：50 蛋币
  - 其他：20 蛋币
- 速度加成：
  - 5分钟内完成：×3
  - 6分钟内完成：×2
  - 7分钟内完成：×1.5
  - 8分钟内完成：×1.2

## 语文挑战

- 题目数量：10 道
- 题目类型：一年级生字拼音选择
- 基础奖励：每题 +5 蛋币
- 完成奖励：+50 蛋币

## 英语挑战

- 题目数量：10 道
- 题目类型：小学词汇朗读识别
- 基础奖励：每题 +8 蛋币
- 完成奖励：+50 蛋币
- 语音识别：使用 Web Speech API
- 容错规则：Levenshtein 距离 ≤ 2 视为正确

## 获取蛋币上限

- 数学挑战满分极速：900 蛋币（300×3）
- 语文挑战全对：100 蛋币
- 英语挑战全对：130 蛋币
